TikTok Content Posting API Integration Demo

📋 App Overview

Tideswell FC is a sports media application that produces short-form soccer highlight content. We integrate with TikTok's Content Posting API to programmatically publish curated highlight clips directly to our TikTok account @tideswellfc.

Integration Flow

INPUT Live match event data (goals, penalties, red cards) via API-Football
↓
PROCESS Auto-generate vertical 9:16 highlight clips with captions & transitions
↓
POST Publish to TikTok via Content Posting API (video.publish scope)

Step-by-Step Demo

1

OAuth Authentication

User authorizes Tideswell FC via TikTok OAuth 2.0. We request the video.publish scope to post content on behalf of the authenticated user.

2

Content Generation

Our pipeline detects match events, sources broadcast footage, and generates a 9:16 vertical highlight clip with animated captions, color correction, and SFX transitions.

3

Video Upload via Content Posting API

We call POST /v2/post/publish/video/init/ to initialize the upload, then chunk-upload the video file and call POST /v2/post/publish/content/ to publish it.

4

Result

The highlight clip appears on the @tideswellfc TikTok account automatically within minutes of a match event.

API Endpoints Used

# Step 1: Initialize upload
POST https://open.tiktokapis.com/v2/post/publish/video/init/
Authorization: Bearer {access_token}
Content-Type: application/json

{
  "source": "PUSH_UPLOAD",
  "title": "⚽ Mbappé Goal — France vs Argentina",
  "privacy_level": "PUBLIC_TO_EVERYONE"
}

# Step 2: Chunk upload video binary
PUT {upload_url}
Content-Range: bytes {start}-{end}/{total}
Content-Type: video/mp4
{video_binary_data}

# Step 3: Publish
POST https://open.tiktokapis.com/v2/post/publish/content/init/

Products & Scopes

Product: Content Posting API

Scope: video.publish
